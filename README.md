# Automated Student Grading System

## What I Did:
Built a dynamic grading sheet for 12 students across 7 core subjects.

## Key Features:
- **Nested IF formulas** to calculate grades (A+, A, B, etc.) based on percentage
- **Conditional Formatting** to automatically highlight marks below 40 in red
- **Professional PDF reporting** layout design

## Tools Used:
Microsoft Excel (Formulas, Data Formatting, Logical Functions)

## Skills Demonstrated:
- Grade Logic Implementation
- Visual Data Alerts
- Report-Ready Formatting
